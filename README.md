# appframe

Generate professional App Store & Play Store promotional screenshots from raw app screenshots.

> Turn simulator captures into polished, store-ready promotional images with device frames, headlines, and styled backgrounds.

## Status

Under active development. See [BUILDING_PLAN.md](./BUILDING_PLAN.md) for the roadmap.

## Features (planned)

- Device frame overlays (iPhone, iPad, Android)
- Multiple template styles (minimal, bold, dark, playful)
- Per-app YAML configuration
- Multi-language screenshot generation
- AI-assisted copy generation via MCP server
- Local web preview with live editing
- Auto-capture from iOS Simulator / Android Emulator
- Direct upload to App Store Connect & Google Play Console

## Quick Start

```bash
npx appframe init
npx appframe generate
```
